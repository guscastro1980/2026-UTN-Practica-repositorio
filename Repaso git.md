## Para que sirve Git

Para versionar nuestro proyecto

## Que es github?

Servicio que permite alojar(hostear) nuestro repositorio en la nube

## Como creo un repositorio con git?

Comando git init, inicializamos nuestro repositorio local

## Como podemos versionar nuestro codigo, es decir guarad los cambios realizados?

git add . (añade al area de preparacion los cambios realizados en el codigo/ el punto gace referencia a traer todos los archivos en el directorio/carpeta raiz)
git commit -m "comentario descriptivo del cambio"
