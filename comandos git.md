## Crear repositorio

git init

## Agragar archivo

git add .

## Versionar o "comitear"

git commit -m "comentario"

## Ver el estado o status actual de nuestro repositorio

git status

## Ver historial de commits

git log

## Renombrar rama maestra a mail

git branch -M main

## Eliminar un archivo (transformarlo en untrucked)

git rm --cached <archivo>

## Añadir origen remoto

git add origin <url exacta del repositorio>

## Revisar origen remoto

git remote -V

## Modificar origen remoto

git remote set-url <url exacta del repositorio>

## Enviar contenido a repositorio en la nube

git push -u origin main
